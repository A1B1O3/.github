# BodyRecord
### 🚩 프로젝트 개요
#### 펜데믹 이후로 늘어난 헬스 인구의 편의성을 높여주기 위한 모바일 서비스를 제작

### 💻 프로젝트 기술 스택


### 💡 프로젝트 주요 기능

* 로그인
  * 구글 소셜 로그인 제공
  * 회원 탈퇴 및 프로필 수정 제공
* 커뮤니티
  * 운동 방법, 방식 공유 제공
* 운동기록
  * 운동을 기록할 수 있는 달력 제공
  * 운동 종류, 횟수 등 상세히 기록 가능한 양식 제공
  * 시간에 따른 체성분 수치 변동 그래프 제공
* 챌린지
  * 챌린지 생성 제공
  * 챌린지 참여 제공
  * 챌린지 인증을 통한 동기부여 제공
